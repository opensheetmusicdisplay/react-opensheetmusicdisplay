# OpenSheetMusicDisplay for React
This repository holds a react component for OpenSheetMusicDisplay. Checkout the ```src/App.js``` file to see how to use it.<br>
<br>
To see the demo, navigate to the project root and do the following `localhost:3000`:
```
npm install

npm start
```
<img src="https://user-images.githubusercontent.com/33069673/79475489-05958480-8008-11ea-9011-2a7227515802.png" height="300">

Right now only some properties are passed through to the library, such as `drawTitle`, `file` and `autoResize`.
